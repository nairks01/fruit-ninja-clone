# Fruit Ninja Clone

A browser-based Fruit Ninja clone with mouse-based gameplay.

This project was developed using the Claude Code CLI on Cursor.

## Features

- Mouse-based fruit slicing gameplay
- Multiple fruit types with different colors
- Combo system for consecutive slices
- Particle effects when fruits are sliced
- Bomb obstacles that end the game
- Responsive canvas-based rendering
- Score tracking

## How to Play

1. Open `index.html` in a web browser
2. Move your mouse to aim
3. Click and hold, then drag to slice fruits
4. Avoid bombs - hitting one ends the game
5. Build combos to increase your score multiplier

## Development

No build process required. Just open `index.html` in a browser to play.

## Game Mechanics

- **Fruits**: Spawn from the bottom of the screen with various colors
- **Slicing**: Draw a line across fruits to slice them
- **Combo**: Each consecutive slice increases the multiplier
- **Bombs**: Black spheres that end the game if sliced
- **Score**: Based on fruits sliced and combo multiplier
